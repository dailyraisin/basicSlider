# Basic Slider Demo

Basic slider demo with pure javascript

## Usage
    basicSlider(el);

## Required Markup
A block element.  The class must match the CSS rules.

Use this for starters: `<div class="slider"></div>`

## Rendered Markup
    <div class="slider">
        <div class="rule"></div>
        <div class="handle"></div>
        <div class="position"></div>
    </div>
